Report Data Processing Activity
The General Data Protection Regulation (GDPR) requires that a register of all data processing activities is kept. This form serves to get an overview of all processes, regardless of these processes using personal data, and to provide insights into which steps could be needed with regards to privacy and information security. The form consists of three parts: general information, privacy, and information security. The form shows which section you are currently in. By using this form, you submit a processing activity (with or without personal data) as a DRAFT to be added to the UU registry. After submitting the form, you will receive an email with additional information. You can follow the link in there to review and alter the information you provided. When the form is definitive, you can submit it as such. When you submit a definitive processing activity, the relevant privacy officers, Data Protection Officer and information security colleagues will be informed. This form is only meant to be used for processes where the UU is responsible for the processing of the data, but where it is not processing it itself. Examples of these processes include: -Research that is entirely conducted at the UU. -Processes where the UU is the secretary/spokesman. -UU business processes. In order to have an appropriate level of information security, it is important to submit every process, even when the process does not contain personal data. This tool will help to determine if a DPIA is needed and which information security measures should be taken ourselves.
